🔍 HOW IT WORKS:

1. User registers/logs in
   ↓
2. Pastes a URL in dashboard
   ↓
3. Backend receives request with auth token
   ↓
4. Checks cache (instant if recent)
   ↓
5. Extracts URL features (protocol, domain, etc.)
   ↓
6. Heuristic analysis (fast pattern matching)
   ↓
7. LLM analysis (if API key available)
   ↓
8. Combines scores → calculates risk (0-100%)
   ↓
9. Saves to database with timestamp
   ↓
10. Returns results to frontend
    ↓
11. User sees:
    - Safety status (Safe/Suspicious/Dangerous)
    - Risk score with visual bar
    - Detailed explanation
    - Security indicators
    - SSL/TLS status
